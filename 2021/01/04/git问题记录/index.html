git问题记录

记录使用git中碰到的问题
commit后发现信息有问题，该怎样修改呢？
执行 git log 查看最近的提交记录
执行 git rebase -i HEAD~3 修改最近3次提交信息
将要修改的日志前的pick改为edit 输入:wq保存
执行 git commit –amend 修改日志，然后:wq保存
修改完成后执行 git rebase –continue
最后强制推送到远程仓库 git push origin-f
